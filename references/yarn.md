# Yarn | Referência Rápida

``` bash
# Instala o pacote Yarn
sudo apt-get update && sudo apt-get install yarn

# Verifica a versão instalada
yarn -v

# Adiciona o pacote Yarn ao projeto (package.json + yarn.lock)
yarn init

# Adiciona pacotes ao projeto
yarn add --dev <nome_do_pacote>
yarn add --dev browser-sync gulp gulp-sass
```
